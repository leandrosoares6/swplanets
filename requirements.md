General purpose: An REST API for manager Star Wars Planets

Entities:
  Planet

Planet attributes:
  String name
  String climate
  String terrain
  Integer numberOfFilmAppearances

Planet use cases:
  Add a planet
  List Planets
  Find by name
  Find by ID
  Destroy planet


Requirements:

1 - Create a planet when passing valid fields
2 - List all persisted planets
3 - Find planet by name
4 - Find planet by ID
5 - Destroy planet
